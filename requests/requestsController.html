1. 查询request列表

查询当前登录用户的request列表

1.1. URL

GET /api/requests/list

1.2. 参数

无

1.3. 返回值

{
    "incomingRequests":[{
        requestInfo...
    }],
    "sendOutRequests":[{
        requestInfo...
    }]
}

2. 按类别查询request列表

按类别查询当前登录用户的request列表

2.1. URL

GET /api/request/list/{type}

2.2. 参数

以下参数在请求路径上提交

  • type
    • incoming
    • sendOut

2.3. 返回值

{
    "content":[],
    "total":0
}

3. 查看请求详情

查询指定请求的详细信息

3.1. URL

GET /api/requests/view/{requestId}

3.2. 参数

以下参数在请求路径传递：

  • requestId：请求ID

3.3. 返回值

{
    senderInfo:{

    },
    recipientInfo:{

    },
    poaInfo:{

    },
    requestInfo...
}

4. 接受或拒绝请求

接受或拒绝某个文档的访问权限请求

4.1. URL

GET /api/requests/process/{requestsId}/{result}

4.2. 参数

以下参数在路径上提交

  • requestsId：请求ID
  • result：接受或拒绝
    • Accepted：接受
    • Declined：拒绝

4.3. 返回值

{
    "data":null,
    "message":"success"
}
