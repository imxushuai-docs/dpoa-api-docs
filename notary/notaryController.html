1. 上传Notary

上传Notary

1.1. URL

POST /api/notary/upload

1.2. 参数

以下参数用json格式以request body上传：

  • type：Document Type，文档类型
  • uploadNotaryFileType：上传的文档类型，有以下值
    • ORIGINAL_DOCUMENT
    • CERTIFIED_NOTARIZED
    • PHOTO_COPY
  • uploaderInfo：对象
    • lastName
    • firstName
    • sex
    • dateOfBirth
    • email
    • phoneNumber
    • address1
    • address2
    • provinceOrTerritory
    • city
    • postalCode
  • witnessList：数组，数组字段如下
    • lastName
    • firstName
    • sex
    • dateOfBirth
    • email
    • phoneNumber
    • address1
    • address2
    • provinceOrTerritory
    • city
    • postalCode
  • createdIn：创建于什么地方
  • NotarizedIn：在什么地方公证
  • fileId：文件id

1.3. 返回值

与上传的数据基本一致。

2. 查询Notary列表

查询notary列表

2.1. URL

GET /api/notary/query

2.2. 参数

  • lastName
  • phoneNumber
  • email
  • dateOfBirth
  • fileId
  • timeOfNotarization
  • notaryPublic
  • page
  • size

2.3. 返回值

{
    "total":10,
    "content":[{
        notaryInfo...
    }]
}

3. 更新notary的状态

更新指定notary的状态值

3.1. URL

POST /api/notary/status

3.2. 参数

以下参数在request body提交

  • notaryId：notary id
  • status：变更后的status值

3.3. 返回值

{
    "data":null,
    "message":"success"
}

4. 按ID查询Notary

按ID查询Notary

4.1. URL

GET /api/notary/query/{notaryId}

4.2. 参数

  • notaryId

4.3. 返回值

{
    "data":{
        notaryInfo
    },
    "message":"success"
}

5. notarize

公证未被公证的公证文件

5.1. URL

POST /api/notary/notarize

5.2. 参数

以下参数在request body中提交

  • notaryId
  • status：固定传 Verified
  • notaryPublic
  • notarizeProvince

5.3. 返回值

{
    "data":null,
    "message":"success"
}
