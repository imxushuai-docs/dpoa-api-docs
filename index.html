2. 接口返回值定义

所有接口均遵守HTTP请求规范，但不严格遵守Http status的返回标准，如：新增不会返回201，还是使用200

2.1. 查询

2.1.1. 列表

  • 参数

    所有的列表查询均采用后端分页，参数传递需要在使用表单传递page以及size字段

  • 返回值

    查询列表的返回值统一以如下格式返回：

    {
    "total":0,
    "content":[{}]
}

2.1.2. 普通查询

  • 参数

    参数参考具体的API

  • 返回值

    {
    "data":{},
    "message":""
}

2.2. 其他接口

  • 参数

    参考具体API

  • 返回值

    {
    "data":{},
    "message":""
}

2.3. 异常情况

所有非200状态码均会使用特殊的返回格式，但http status会根据不同类型的错误而不同，如果参数错误的异常会是400等。

  • 异常返回格式

    {
    "data":{},
    "message":""
}
